# doctor-credentialing-app
Doctor Credentialing Application

## Angular Highlights
* Authentication Guards
* i18n - Internationalization / Localization using locale property from JWT
* Setting page title
* CRUD using modal
* Form Validations

## Default Accounts
* Admin
    * U: admin@email.com
    * P: Password@123
* Approver
    * U: approver@email.com
    * P: Password@123
